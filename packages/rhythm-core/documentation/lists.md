## Unordered list
```html example
<ul>
    <li>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</li>
    <li>Morbi rutrum sapien non arcu tempus hendrerit vitae eu nisi.</li>
    <li>Vestibulum dictum sapien vitae est cursus, sit amet eleifend tellus scelerisque.</li>
</ul>
```



## Ordered list
```html example
<ol>
    <li>Ut vulputate odio id libero vestibulum fermentum.</li>
    <li>Phasellus at sapien ut turpis faucibus commodo ac et nibh.</li>
    <li>Donec ut velit nec lectus vehicula auctor eu sed ipsum.</li>
</ol>
```

## Mixed lists
```html example
<ul>
    <li>Donec ac dui eget lorem feugiat luctus.</li>
    <li>Donec consequat turpis quis odio commodo, sed accumsan magna vulputate.
        <ol>
            <li>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</li>
            <li>Morbi rutrum sapien non arcu tempus hendrerit vitae eu nisi.</li>
        </ol>
    </li>
    <li>Quisque in nisl lobortis, pharetra massa quis, pulvinar ante.</li>
    <li>Sed id ante a ex commodo rutrum.</li>
</ul>
```

```html example
    <dd>Sed efficitur malesuada placerat. Nam aliquet eros sed scelerisque finibus. Aenean ipsum libero.</dd>
    <dt>Nulla consequat</dt>
    <dd>Vivamus posuere facilisis purus a pulvinar.</dd>
    <dt>Vivamus</dt>
    <dd>Aenean non molestie eros.</dd>
</dl>
```
